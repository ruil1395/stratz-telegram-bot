# Stratz Dota 2 Telegram Bot

Бот для получения статистики Dota 2 с сайта Stratz.com

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать `.env` файл с токенами
4. Запустить: `python bot.py`

## Команды

- `/start` - Главное меню
- `/player &lt;Steam ID&gt;` - Статистика игрока
- `/match &lt;Match ID&gt;` - Информация о матче
- `/heroes` - Список героев
