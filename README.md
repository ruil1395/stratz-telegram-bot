# 🤖 Stratz Dota 2 Telegram Bot

Telegram бот для получения статистики Dota 2 с сайта [Stratz.com](https://stratz.com)

## 🎮 Возможности

- 📊 Статистика игроков (JSON/CSV)
- 🎮 Детали матчей в CSV
- 🏆 Список про-игроков
- 📁 Выгрузка данных в файлы
